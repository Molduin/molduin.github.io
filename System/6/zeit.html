Zeitablauf und Tagesplanung

Zeiteinheiten

Ein Tag im Spiel ist eingeteilt in 8 Zeiteinheiten bzw. Tageszeiten, die dementsprechend je circa 3 Stunden lang sind. Die meisten Aktivitäten, denen Charaktere oder Gruppen im Laufe des Tages nachgehen können, nehmen eine oder mehrere dieser Tageszeiten ein.

Schlaf

Das Namenlose System™ erlaubt es den Spielern, den Chronotypen ihres Charakters zu wählen.

Jeder Charakter erhält eine Anzahl an "Schlafwürfeln", die 10 plus der Anzahl seiner Willenswürfel entspricht. Diese kann er auf die 8 Tageszeiten verteilen, hierbei muss jedoch auf jede Tageszeit mindestens ein Schlafwürfel gesetzt sein.
Eine äquivalente Rechnung: Jeder Charakter erhält eine Anzahl an Schlafwürfeln, die 2 plus der Anzahl seiner Willenswürfel entspricht, und kann diese beliebig verteilen. Zusätzlich wird auf jede Tageszeit ein zusätzlicher Schlafwürfel gestezt.

Schläft ein Charakter, so werden die Schlafwürfel, die auf die Zeiten, die er verschläft, gesetzt wurden, zusammengenommen und geworfen: Zeigt mindestens einer der Würfel eine 6, ist der Charakter ausgeschlafen.

Die Bedingungen, unter denen ein Charakter schläft, können die Anzahl der Würfel, die geworfen werden, beeinflussen, indem Würfel insgesamt oder pro Zeiteinheit hunzugenommen oder abgezogen werden oder indem sie z.B. gesechstelt werden.

Für jede Zeiteinheit, die ein Charakter ununterbrochen wach ist, erhält er eine Anzahl "Müdigkeitswürfel", die der Anzahl an Schlafwürfeln auf dieser Zeiteinheit entspricht.
