Glossar

Proben

  • Eine Probe auf (a | b) bedeutet, dass zwei Würfe ausgeführt werden: Die Probe gelingt, wenn das Ergebnis des ersten Wurfs <= a ist, oder wenn das Ergebnis des zweiten Wurfs <= b ist.
  • (a | b) + c ("eine um c erleichterte Probe") ist gleichbedeutend mit (a + c | b + c).

  • MobMal: Mobilitätsmalus der Rüstung

Kampfsystem

Waffenskills

Nahkampfskills

Skillpunkte (SP) müssen auf eine der folgenden Waffenklassen gesetzt werden: Dolch, Einhänder, Zweihänder, Lang. Auf eine Waffenklasse gesetzte SP können im Umgang mit allen Waffen dieser Waffenklasse genutzt werden.

Nahkampf

In einem Duell versuchst du, dir durch verschiedene Aktionen (Parieren, Blocken, Ausweichen, Finten, Angreifen...) einen Vorteil zu verschaffen. Bei genügend Vorteil verliert dein Gegner einen Zug. Die Skala hierfür, auch "Tempo" genannt, sieht wie folgt aus:

Gegner verliert Zug <-- Vorteil <--> Ausgeglichen <--> Nachteil --> Du verlierst Zug

Beim Start eines Duells ist das Tempo "ausgeglichen", und jedes Mal, wenn ein Duellant seinen Vorteil ausspielt, d.h. das Tempo den Rand der Skala erreicht hat, wird es auf "ausgeglichen" zurückgesetzt.

Aktionen, die Tempo generieren:

  • Ein misslungener Angriffswurf: 2 --->
  • Eine erfolgreiche Parade: <--- 1
  • Ein vorhersehbarer Angriff: <--- 1 für Verteidiger

Ausdauerpunkte und Erschöpfung

Deine Ausdauerpunkte (AP) entsprechen dem zwanzigfachen deiner Ausdauer. Dein Charakter hat einen Erschöpfungswert (normalerweise 0), der sich jedes mal um 1 erhöht, wenn deine AP auf 0 fallen (AP werden wieder komplett aufgefüllt, sobald sie 0 erreichen). Für jede Stufe der Erschöpfung bekommst du einen Malus von -2 auf alle Würfe. Jeder Charakter regeneriert einen AP am Ende jedes Zuges. Eine kurze Rast verringert deine Erschöpfung um 1.

Kraftpunkte (KP)

Pro Runde erhält jeder Charakter, pro Hand, eine Anzahl von Kraftpunkten (KP), die zwei mal seiner Kk entspricht. Für zweihändig geführte Objekte erhalten Charaktere insgesamt 3Kk KP. Kraftpunkte sind nötig, um anzugreifen, zu blocken und zu parieren. Für jeden genutzten KP verlierst du einen AP. Nicht genutzte KP verfallen am Ende der Runde.
